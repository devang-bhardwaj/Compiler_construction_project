Product Requirements Document (PRD) + System Architecture

1. Product Overview
Product Name

Intermediate Code Optimizer (ICO)

Vision

Build a developer tool that analyzes intermediate code (IR / three-address code / pseudo-IR) and performs compiler optimization techniques while providing interactive visualization and explanations.

Problem Statement

When students or developers study compilers, they usually struggle to understand optimization techniques applied on intermediate representations such as:

Constant Folding

Dead Code Elimination

Common Subexpression Elimination

Strength Reduction

Most compiler tools run these optimizations silently.

The Intermediate Code Optimizer provides:

✔ Visual transformation of code
✔ Step-by-step optimization passes
✔ Graphical control flow visualization
✔ Performance metrics comparison

Features:

Syntax highlighting

4.2 Optimization Engine

The system will perform optimization passes.

Passes
Optimization	Description
Constant Folding	Evaluate constant expressions
Constant Propagation	Replace variables with known constants
Dead Code Elimination	Remove unused instructions
Common Subexpression Elimination	Avoid repeated calculations
Strength Reduction	Replace expensive operations
Copy Propagation	Replace redundant copies
4.3 Optimization Pipeline

Users can run:

Single optimization

Full pipeline

Example pipeline:

Generate Optimized IR

8. Detailed System Design
IR Parser

Responsibilities:

Parse code

Tokenize instructions

Build instruction objects

Example structure:

   op
   arg1
   arg2

Basic Block Builder

Groups instructions.

Example
